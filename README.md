# 頻道說明
出門在外
除了手機錢包鑰匙要帶好之外
最常要找的就是廁所了!

因此「憋急。別急」誕生了
只要傳送所在位置
即可回復你鄰近可徒步到公廁
讓你內急時不會再從容失措
